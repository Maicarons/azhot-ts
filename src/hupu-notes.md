# 虎扑爬虫说明

## 实现详情

根据提供的Go代码创建了虎扑爬虫，实现了以下功能：

- **平台标识**: hupu
- **显示名称**: hupu
- **图标URL**: https://www.hupu.com/favicon.ico
- **API端点**: https://www.hupu.com/
- **数据结构**: 包含体育资讯标题和链接

## 代码实现

实现了与Go代码逻辑一致的TypeScript版本，包括：

1. 使用fetch API获取网页内容
2. 设置适当的请求头信息（User-Agent、Referer等）
3. 10秒超时机制
4. 使用正则表达式匹配虎扑页面的体育资讯结构
5. 数据格式转换为HotItem接口
6. URL路径正确拼接处理（确保相对URL转换为绝对URL）

## 测试结果

- API端点 `/hupu` 返回正确的数据格式
- 成功返回了29个体育资讯条目
- 图标、名称和URL与Go代码完全一致
- 体育资讯标题和链接正确解析
- 相对URL被正确转换为绝对URL
